&emsp;&emsp;日本某地发生了一件谋杀案，警察通过排查确定杀人凶手必为4个嫌疑犯的一个。以下为4个嫌疑犯的供词。
<!--more-->
A说：不是我。
B说：是C。
C说：是D。
D说：C在胡说。
已知3个人说了真话，1个人说的是假话。
现在请根据这些信息，写一个程序来确定到底谁是凶手。
&emsp;&emsp;已知3个人说了真话，1个人说的是假话。只需要把他们说的话转换为逻辑表达式，如果相加为3，则说明找到了凶手。
代码如下
```c 
#include<stdio.h>
#include<stdlib.h>
int main()
{
	int man;
	for (man = 'a'; man <= 'd'; man++)
	{
		if (((man != 'a') + (man == 'c') + (man == 'd') + (man != 'd')) == 3)
		{
			printf("凶手是%c\n", man);
			break;
		}
	}

	system("pause");
	return 0;
}
```
